행복의 조건 & 이유 5가지로 정의해보자

행복은 S + C + V  
S : Set Point(주어진 환경. 유전적. 내면적 등)  
C : Condition(외부적 요인, 돈, 좋은직장, 친구)  
V : Voluntary Actions(봉사, 취미, 활동 등)

모두의 예상과 달리 S : C : V = 50 : 10 : 40
