# 진법 변환

10진수 <----> 2^n진수 변환 방법

```  
1) 10진수 수를 2진화 시킨다.  
2) 2진수를 n개씩 묶는다.  
3) 갯수가 모자란 경우, 앞에 0을 붙여 맞춘다.  
4) 묶은 숫자를 한 자리씩 적는다.  
```


ex) 10진수 555 표현

**555 = 1000101011(2)**

```
**4진수**로 555 = 20223  
10 00 10 10 11  
 2  0  2  2  3  
```

```
**8진수**로 555 = 1053  
001 000 101 011  
  1   0   5   3  
```

```
**16진수**로 555 = 22b  
0010 0010 1011  
   2    2   11  
```
